# Sistema Integrado de Monitoramento de Gás

## Descrição
Este repositório reúne dois subprojetos complementares voltados à segurança e ao monitoramento do uso de gás de cozinha.

O sistema é dividido em duas frentes principais:

1. **Medidor de Nível de Gás**
   - responsável por estimar a quantidade de gás restante no botijão

2. **Detector de Vazamento de Gás**
   - responsável por identificar possíveis vazamentos no ambiente

## Objetivo Geral
Desenvolver uma solução integrada capaz de aumentar a segurança e a previsibilidade no uso do gás de cozinha em ambientes domésticos.

## Estrutura do Repositório

### Subprojeto 1 — Medidor de Nível
Pasta: `projeto-medidor-nivel/`

Responsável por:
- leitura do peso do botijão
- estimativa de nível de gás
- alertas de gás baixo

### Subprojeto 2 — Detector de Vazamento
Pasta: `projeto-detector-vazamento/`

Responsável por:
- monitoramento da presença de gás no ambiente
- detecção de vazamento
- alertas de segurança

## Integração
Os dois subprojetos fazem parte de uma proposta integrada, podendo compartilhar:
- documentação
